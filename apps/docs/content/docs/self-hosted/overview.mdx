## What you operate

```
┌───────────────────────────────────────────────────────────────┐
│  Your infrastructure (sovseal sees none of it)                │
│  ┌──────────────────┐    ┌─────────────────────────────────┐  │
│  │  Edge function   │    │  Postgres (+ pgvector optional) │  │
│  │  v2-agent-state  │───►│  • replications table           │  │
│  │  (Hono / Deno)   │    │  • projects table               │  │
│  └──────────────────┘    └─────────────────────────────────┘  │
│           ▲                                                   │
│           │ HTTPS, ciphertext only                            │
└───────────┼───────────────────────────────────────────────────┘
            │
   ┌────────┴────────┐
   │  Your agents    │  (local SDK / MCP server)
   └─────────────────┘
```
