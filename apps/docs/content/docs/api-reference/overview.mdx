## Endpoint catalog

| Method | Path                    | Purpose                              |
| ------ | ----------------------- | ------------------------------------ |
| POST   | `/v2/replicate/push`    | Upload one or more ciphertext records |
| POST   | `/v2/replicate/pull`    | Fetch records by path-hash since cursor |
| GET    | `/v2/replicate/list`    | List path-hashes for a project       |
| POST   | `/v2/projects`          | Create a project (Platform only)     |
| POST   | `/v2/projects/:id/keys` | Rotate or revoke an API key          |
