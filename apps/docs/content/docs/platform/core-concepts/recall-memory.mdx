## Quick reference

- **Latency**: p50 6.1 ms · p95 10.4 ms · p99 21.8 ms (10K records, warm cache)
- **Cold start**: ~1.2 s on first call (embedding model + index warmup)
- **Network**: 0 RTT — calls never leave the device
- **Cache**: 256-entry LRU on query embeddings (configurable)
