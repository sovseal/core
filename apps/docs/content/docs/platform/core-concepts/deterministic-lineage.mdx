## Lineage as a DAG

```
   snapshot₀ ── snapshot₁ ── snapshot₂ ── snapshot₃ (HEAD)
       │            │            │
       │            │            └─ store_memory("strategy.v3", …)
       │            └─ store_memory("strategy.v2", …)
       └─ store_memory("strategy.v1", …)

   rollback(snapshot₂)  →  HEAD := snapshot₂      (no data lost; new HEAD pointer)
   fork(snapshot₁)      →  branch with parent₁    (multi-agent divergent state)
```
