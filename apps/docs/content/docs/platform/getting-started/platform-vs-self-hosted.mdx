require you to own the ciphertext store.
</Callout>

## At a glance

|                       | Platform (Managed)                  | Self-Hosted                             |
| --------------------- | ----------------------------------- | --------------------------------------- |
| **Read latency**      | 0 RTT (always local)                | 0 RTT (always local)                    |
| **Write replication** | Managed, write-behind               | You operate the endpoint                |
| **Storage**           | Managed Postgres + object store     | Your Postgres + object store            |
| **Encryption**        | Client-side AES-256-GCM             | Client-side AES-256-GCM (configurable)  |
| **Key custody**       | Device-only (server-blind)          | Device-only (server-blind)              |
| **Ops surface**       | None                                | One edge function + one Postgres schema |
