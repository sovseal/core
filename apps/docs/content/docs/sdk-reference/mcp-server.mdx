## Tools exposed

| Tool             | Purpose                              | Network |
| ---------------- | ------------------------------------ | ------- |
| `store_memory`   | Persist a record under a path        | 0 RTT (write-behind) |
| `recall_memory`  | Semantic search the local index      | 0 RTT |
| `list_memories`  | Enumerate stored paths (with filter) | 0 RTT |
| `delete_memory`  | Remove a record + tombstone replica  | 0 RTT (tombstone replicated) |
